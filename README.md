# Scoop Bucket for obsi

[Scoop](https://scoop.sh) bucket for [obsi](https://github.com/avi8or/obsidian-cli-quick-notes) - Obsidian CLI with Quick Notes support.

## Installation

```powershell
# Add this bucket
scoop bucket add obsi https://github.com/avi8or/scoop-obsi

# Install obsi
scoop install obsi
```

## Update

```powershell
scoop update obsi
```

## Uninstall

```powershell
scoop uninstall obsi
```
